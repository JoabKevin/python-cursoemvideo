# Operadores aritméticos

* *+* = Adição
* *-* = Subtração
* (*) = Multiplicação
*  /  = Divisão
* **  = Potência
* //  = Divisão inteira
*  %  = Módulo (resto da divisão)

## Símbolos
* =  - Atribuição
* == - Igualdade

## Ordem de precedência
* 1 - ()
* 2 - **
* 3 - *, /, //, % por ordem de chegada
* 4 - +, -
